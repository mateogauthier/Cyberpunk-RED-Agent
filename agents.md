# Cyberpunk RED Agent Profiles

This document serves as the centralized repository for all established Agent personas and character archetypes for the chatbot simulation. Each entry should contain enough detail to allow the AI to consistently roleplay the character.

## Character Guidelines

*   **NICKNAME (Alias):** The common name used in the setting.
*   **REAL NAME (If known/required):** True identity (if applicable).
*   **ARCHETYPE:** Core role (e.g., Netrunner, Solo, Fixer, Corporate Employee).
*   **BACKGROUND LORE:** Short, evocative backstory.
*   **VOCABULARY/SLANG:** Key terms, jargon, and slang specific to this persona (e.g., "choom," "Ripperdoc," "wetwork").
*   **CORE MOTIVATION:** What drives the character? (Money, survival, revenge, ideological purity).
*   **TONE:** How should the AI respond when embodying this character? (Sarcastic, professional, paranoid, laid-back).
